# Sons pour GymAI

## Notification de fin de repos

Pour activer le son de notification quand le temps de repos est terminé :

1. Ajoutez un fichier son nommé `rest_complete.mp3` dans ce dossier
2. Le fichier peut être au format MP3, WAV, ou autre format audio supporté
3. Recommandé : un son court (0.5-1s) type "ding" ou "beep"

## Exemple de sources pour télécharger des sons gratuits

- [Freesound.org](https://freesound.org) - Sons gratuits sous licence Creative Commons
- [Zapsplat.com](https://www.zapsplat.com) - Effets sonores gratuits
- [Mixkit.co](https://mixkit.co/free-sound-effects/) - Sons gratuits

### Recherches suggérées
- "notification bell"
- "ding sound"
- "timer complete"
- "workout bell"

## Note technique

Le fichier doit être nommé exactement `rest_complete.mp3` pour être détecté par l'application.
Si aucun fichier n'est présent, l'app affichera uniquement une notification visuelle (SnackBar).
